# Google Ads Scripts

Soy **Arnau**, Performance Media Buyer. Aquí tengo los scripts que uso en el día a día para no volverme loco con Google Ads. Automatizaciones, análisis de datos, cosas que ahorro tiempo haciendo en Python en vez de clickar en la UI.

Estudio el Máster en Data Analytics en Nuclio. Estos scripts vienen de problemas reales, nada teórico.

## Contenido

### Google Ads Automation
- `campaign_creator.py` - Creación automática de campañas desde CSV
- `rsa_generator.py` - Generación de Responsive Search Ads
- `keyword_clustering.py` - Clustering semántico de keywords

### Analysis & Reporting
- `placement_heatmap.py` - Análisis de rendimiento de placements
- `correlation_analysis.py` - Correlaciones entre métricas
- `performance_metrics.py` - Cálculo de KPIs

### Data Processing
- `data_cleaning.py` - Limpieza y normalización de datos
- `export_handler.py` - Lectura/escritura de exports de Google Ads
- `bigquery_connector.py` - Integración con BigQuery

## Stack

Python · Pandas · NumPy · scikit-learn · Plotly · BigQuery

## Setup

```bash
pip install -r requirements.txt
python script.py
```
